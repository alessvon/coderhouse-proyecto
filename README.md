# Portfolio Personal — Coderhouse

Proyecto de portfolio personal desarrollado con **HTML5**, **CSS3** y **Bootstrap 5**.

## Estructura

```
coderhouse-proyecto/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── README.md
```

## Secciones

- **Hero** — Presentación principal
- **Sobre mí** — Información personal
- **Habilidades** — Stack tecnológico
- **Proyectos** — Cards de proyectos
- **Contacto** — Formulario de contacto

## Tecnologías

- HTML5
- CSS3 (variables, flexbox, grid)
- Bootstrap 5.3
- Bootstrap Icons
- JavaScript (vanilla)

## Cómo usar

Abrí `index.html` en tu navegador. No requiere instalación ni servidor.
